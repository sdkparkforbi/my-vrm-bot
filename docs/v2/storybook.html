준비물 챙기기계정 3개 + 프로그램 1개

🧰 🐙 ▲ 💛
  1. Git 설치 — 코드를 옮기는 택배 기사 (다운로드)
  2. GitHub 가입 — 내 코드를 보관하는 창고 (github.com)
  3. Vercel 가입 — 내 봇을 인터넷에 띄워 주는 무대 (vercel.com)
  4. 카카오 디벨로퍼스 가입 — 로그인·공유 연결 (developers.kakao.com)
🧑‍🏫 비유하자면… GitHub은 창고, Vercel은 무대, Git은 창고와 무대 사이를 오가는 택배 기사예요.

내 봇 복제하기템플릿 → 내 저장소

🧩 ➜ 📥 ➜ 📤

준비된 봇 템플릿을 그대로 복사해서 내 GitHub 창고에 올립니다.

  1. GitHub에서 빈 저장소 만들기 (이름: my-bot, Public, 체크박스 모두 OFF)
  2. 내 컴퓨터에서 아래를 차례로 (까만 창에 한 줄씩):
git clone https://github.com/sdkparkforbi/my-vrm-bot.git .
rmdir /s /q .git
git init -b main
git add . && git commit -m "내 봇 시작"
git remote add origin https://github.com/[내아이디]/my-bot.git
git push -u origin main
⚠️ 마지막 git remote add 의 주소는 1번에서 만든 내 저장소 주소예요 (템플릿 주소 아님).
🚀

인터넷에 올리기Vercel 배포 + 내 닉네임

📦 ➜ ▲ ➜ 🌐

Vercel → New Project → 내 저장소 선택 → 환경 변수 입력 후 Deploy.

  1. 닉네임(BOT_ID) 정하기 — 영문 소문자·숫자·하이픈, 예: bungae
  2. 환경 변수에 같은 값 2개: BOT_ID 와 VITE_BOT_ID
  3. 카카오 키 VITE_KAKAO_JS_KEY (Step 6에서 발급, 지금은 임시값 OK)
  4. Deploy → 1~3분 후 https://my-bot.vercel.app 같은 내 봇 주소 완성! 🎉
🧑‍🏫 닉네임이 곧 내 봇의 이름표예요. 이 이름으로 봇이 내 지식·성격을 찾아옵니다. 친구 봇과 절대 안 섞여요.
💛

카카오 연결로그인 + 친구에게 공유

💛 ➜ 🔑 ➜ 📲
  1. Kakao Developers → 앱 만들기 → JavaScript 키 복사
  2. 플랫폼 → Web 에 내 Vercel 주소 등록
  3. 카카오 로그인 ON + Redirect URI 설정
  4. Vercel 환경변수 VITE_KAKAO_JS_KEY 를 진짜 키로 교체 → Redeploy
💚 키는 코드에 적지 않고 환경변수로 넣어요. 그래서 안전합니다.

지식·성격 넣기 ⭐이게 봇을 똑똑하게 만드는 핵심

📚 ➜ 🤖 ➜ 💡

내 봇 사이트의 /rag 페이지(또는 화면 오른쪽 위 🧠 아이콘)에서 직접 관리해요. 로그인 후 내 닉네임으로 저장됩니다.

🎭
페르소나 (성격·말투)
"너는 친근한 반말로 설명하는 ○○ 봇이야" 처럼 자유 서술 → 저장
▼
✨
지식 (자동 청크)
텍스트(노트·기사·위키)를 붙여넣고 "청크 생성" → AI가 Q&A로 정리 → 저장
▼
📚
목록 확인
등록된 지식이 쌓이는 걸 보고, 필요 없으면 삭제
🧑‍🏫 비유하자면… 페르소나는 봇의 성격, 지식은 봇이 공부한 교과서예요. 둘 다 넣어야 똑똑하고 나다운 봇이 됩니다.
💚 텍스트는 한 번에 500~5,000자가 적당해요. 붙여넣고 30초~1분 기다리면 자동으로 정리됩니다.
