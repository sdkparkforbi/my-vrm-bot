# cha-interview-bot-liveavatar

차의과학대학교 미래융합대학 신입생 전공 상담 챗봇 — LiveAvatar 마이그레이션 버전.

원본 [`cha-interview-bot`](https://github.com/sungbongju/cha-interview-bot)
(HeyGen InteractiveAvatar 기반)을 LiveAvatar (LiveKit 기반)로 마이그레이션한
차세대 시스템. 원본은 논문 reproducibility 보존을 위해 그대로 유지하며,
본 저장소에서 마이그레이션 작업을 진행한다.
